# Teste para a Vaga de Engenheiro de IA

## Papel
Você é um engenheiro de IA sênior especializado em criação de agentes e multiagentes.

## Objetivo
Sua missão é propor soluções para o desenvolvimento de agentes.

## Contexto
Estou anexando um documento chamado "AI Developer Agent Real Estate Task.pdf" referente a um teste de uma vaga para engenheiro de IA numa startup do setor imobiliário. O objetivo é criar um sistema multiagentes de gerenciamento de ativos e propriedades para uma empresa imobiliária com Python e LangChain.

Além disso, estou anexando também um arquivo chamado "cortex.parquet" que possui um catálogo de propriedades, o qual será usado no projeto para que o agente possa consultar.

## Instruções
Execute as seguintes instruções:
1. Leia atentamente o enunciado do documento "AI Developer Agent Real Estate Task.pdf", com todos os pontos citados, interprete o texto e traduza para a língua portuguesa do Brasil.
2. Proponha a estrutura de uma solução mínima para o problema em questão, levando em consideração as boas práticas de desenvolvimento.
3. Apresente uma primeira solução num Notebook Python com uma ordem lógica e breve explicação sobre cada etapa da solução.
4. Crie um passo a passo para a criação de um projeto real seguindo uma estrutura comum para soluções agênticas, onde os códigos sejam modularizados.
5. Crie um arquivo README.md seguindo o que foi requisitado na tarefa.
6. Por fim, crie um passo a passo para a criação de um repositório no GitHub com o projeto criado.

## Considerações
Considere os seguintes pontos:
* O planejamento proposto da solução (passo 2) deve estar em Português do Brasil.
* O notebook Python e o projeto modularizado com o arquivo README.md (passos 3, 4 e 5) devem estar em inglês, incluindo os comentários e a documentação do código.
* A solução deve conter APENAS o que o enunciado pede, sem a inclusão de melhorias ou requisitos fora do escopo.
